# 2D Graphics Editor

A C-based 2D graphics editor for creating and manipulating graphics.

## Features

- Draw and manipulate 2D shapes
- Interactive graphics rendering
- Cross-platform support

## Getting Started

### Prerequisites

- C compiler (gcc, clang, or MSVC)
- Graphics libraries (SDL2, OpenGL, or similar)

### Building

```bash
gcc graphics_editor.c -o graphics_editor
```

### Running

```bash
./graphics_editor
```
